--------------------
FiveStarRating
--------------------
Author: Alexander Maltsev <admin@itchief.ru>
--------------------

A five star rating system for resources MODX Revolution.

Designed for not authorized users, i.e. the guests. Protection against cheating
is based on IP and LocalStorage. Disable security on IP performs by setting the
system parameter simplerating_ip value "No".

How to use:

1. For the current resource

[[!SimpleRating]]

2. For the specified resource

[[!SimpleRating? &id=`3`]]

3. With an essential design

[[!SimpleRating? &tpl=`tplSimpleRating1`]]
